# Proyecto de implementación de carros eléctricos en perú 

En este proyecto se busca entender que tan factible resulta implementar carros electricos en perú de manera similar a los que se tienen en USA,
donde los carros electricos cuenta con propias estaciones de carga.

Primero se trabajará el modelo utilizando datos de otros país, especificamente [Dataset de viajes Uber – Kaggle](https://www.kaggle.com/datasets/ruchikakumbhar/uber-dataset/data).
